# X-Fofa
基于Fofa会员前提，调用API，超过1w条不会扣F币

![](./cmd.png)



### 用法
* Usage: python3 X-Fofa.py -q 'app = xxx' -o result.txt
* Usage: python3 X-Fofa.py -q 'app = xxx' -all 1 -o result.txt
